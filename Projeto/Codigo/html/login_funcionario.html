Para novos funcionários as senhas padrões são feitas pelas 3 primeiras letras do nome + dia do aniversário + 3 ultimos digitos do cpf + 2 primeiras letras do primeiro sobrenome.
Exemplo: João Pereira Cruz | Nasc: 15/02/1995 | CPF: 910.438.604-31 a senha é joa15431pe
